# 투자 포트폴리오 트래커 설치 가이드

## 구조

```
portfolio_app/
  ├── Code.gs      ← Google Apps Script 서버 코드
  ├── index.html   ← 웹앱 UI
  └── 설치가이드.md
```

---

## 설치 방법 (5분)

### 1단계 — 새 Google Sheets 파일 만들기

1. [Google Sheets](https://sheets.google.com) 에서 **새 스프레드시트** 생성
2. 파일 이름: `투자포트폴리오` (원하는 이름으로)

### 2단계 — Apps Script 열기

1. 메뉴 → **확장 프로그램 → Apps Script** 클릭
2. 좌측에 `코드.gs` 파일이 있을 텐데, **전체 내용을 삭제**

### 3단계 — Code.gs 붙여넣기

1. `portfolio_app/Code.gs` 파일 내용 **전체 복사**
2. Apps Script 에디터에 **붙여넣기**
3. **저장** (Ctrl+S)

### 4단계 — index.html 추가

1. Apps Script 에디터 좌측 **파일 추가(+)** 버튼 클릭
2. **HTML** 선택
3. 파일명: `index` (확장자 없이)
4. `portfolio_app/index.html` 내용 **전체 복사 후 붙여넣기**
5. **저장**

### 5단계 — 시트 초기화 실행

1. Apps Script 에디터에서 함수 목록에서 `setupSheets` 선택 후 ▶ 실행
2. 권한 요청 팝업 → **검토 → 허용**
3. 완료 메시지 확인 → Google Sheets에 5개 시트 자동 생성됨

### 6단계 — 웹앱 배포

1. Apps Script 에디터 우측 상단 → **배포 → 새 배포**
2. 유형: **웹 앱**
3. 설정:
   - 설명: `투자 포트폴리오 트래커`
   - 다음 사용자로 실행: **나**
   - 액세스 권한: **나만 (또는 조직 내)**
4. **배포** 클릭
5. 생성된 **웹앱 URL** 복사 → 브라우저에서 접속!

## 생성되는 시트 구조

| 시트명 | 설명 |
|--------|------|
| `설정_계좌` | 계좌 목록 (계좌ID, 계좌명, 증권사, 자산종류) |
| `종목현황` | 보유 종목 (계좌별, 매수 시 자동 업데이트) |
| `거래내역` | 매수/매도 기록 |
| `입금내역` | 입출금 기록 |
| `배당내역` | 배당 수령 기록 |

## 사용법

### 처음 시작
1. **설정** 페이지 → **계좌 추가**
   - 예: `토스증권 US` / 미국주식
   - 예: `업비트` / 코인
2. **거래내역** → **+ 거래 입력** (매수 입력 시 종목현황 자동 반영)

### 현재가 업데이트
- **대시보드** → 종목별 현재가($) 직접 입력 → **적용** 버튼
- 환율도 함께 입력하면 원화 환산 자동 계산

### 계좌별 수익률 확인
- **계좌별 현황** 페이지에서 계좌 카드 클릭
- 해당 계좌의 종목 목록 및 수익률 확인
