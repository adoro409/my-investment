# 설치방법 — 딱 3단계

## 준비물
- Google 계정 (이미 있음 ✅)
- 새 Google Sheets 파일 1개

---

## 1단계 — Apps Script 열기

1. [sheets.google.com](https://sheets.google.com) → **새 스프레드시트** 만들기
2. 메뉴 상단 → **확장 프로그램** → **Apps Script** 클릭

> 💡 새 탭에 코드 에디터가 열립니다

---

## 2단계 — 코드 붙여넣기

1. 에디터 좌측에 `코드.gs` 파일이 있음
2. 기존 내용 **전체 선택(Ctrl+A) → 삭제**
3. `installer.gs` 파일 내용 **전체 복사 → 붙여넣기**
4. **저장 (Ctrl+S)**

---

## 3단계 — install() 실행

1. 상단 함수 선택 드롭다운에서 **`install`** 선택
2. ▶ **실행** 버튼 클릭
3. 권한 요청 팝업 → **권한 검토 → 계속 → 허용**
4. 완료 팝업 확인 ✅ → 구글 시트에 5개 시트 자동 생성됨

---

## 웹앱 URL 받기 (마지막)

1. Apps Script 오른쪽 상단 → **[배포]** → **[새 배포]**
2. 유형: **웹앱**
3. 다음 사용자로 실행: **나**
4. 액세스 권한: **나만**
5. **[배포]** 클릭
6. 나온 **URL 복사** → 브라우저 즐겨찾기 추가

🎉 **완료! 이제 URL 접속하면 앱 사용 가능**

---

## 이후 업데이트 시

코드를 수정하고 싶을 때:
- Apps Script 에디터에서 수정
- **[배포]** → **[배포 관리]** → 연필 아이콘 → **새 버전** → 배포

---

## 생성되는 시트

| 시트 | 용도 |
|------|------|
| 💳 계좌목록 | 계좌 등록 |
| 📈 종목현황 | 보유 종목 (거래 입력 시 자동 반영) |
| 🔄 거래내역 | 매수/매도 기록 |
| 💰 입출금 | 입출금 내역 |
| 🎁 배당내역 | 배당 수령 기록 |
